Métodos GET e POST

Dentro do HTML, existem dois métodos de formulários para recebimento de dados. São eles:

  • GET: Exibe o preenchimento do formulário na pesquisa
  • POST: Oculta o metodo de pesquisa na URL.

OBS: É importante verificar que, o POST não é 100% seguro, pois através do "inspecionar elemento", é possível localizar os dados preenchido. Por isso, existe a necessidade de criptografia das informações através de https

Quando usar GET e POST?

Quando os dados preenchidos não forem sensíveis, pode ser usado o método GET, porém se forem, então utiliza-se o POST

IMPORTANTE!!

O método GET possui limite de até 3 mil bytes que é equivalente a 3 mil letras. Além de não poder ser utilizado para envio de fotos.
